# Control Flow

## If/Else

```rs
fn main() {
    let score = 85;

    if score >= 60 {
        println!("Pass");
    } else {
        println!("Fail");
    }
}
// Pass
```

```rs
fn main() {
    let age = 20;
    let can_vote = if age >= 18 { "Yes" } else { "No" };
    println!("Can vote? {can_vote}");
    // Can vote? Yes
}
```
